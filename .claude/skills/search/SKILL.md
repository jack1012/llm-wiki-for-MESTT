# search 技能：快速知識定位

## 核心功能
透過讀取檔案內容，快速定位關鍵字在 Wiki 中的分布，優於單純閱讀 `index.md`。

## 使用場景
- 跨文獻尋找特定的「作者」提及。
- 尋找某個「理論」在不同論文中的解釋。
- 檢查某個「術語」是否已經存在或有衝突。

## 執行流程
1. 接收關鍵字。
2. 嘗試動態定位 `qmd` 或直接呼叫 `npx @tobilu/qmd`。
3. 執行指令：
   `npx @tobilu/qmd search "<關鍵字>"`
   （若為語義查詢則使用 `query` 指令）。
4. 在對話中呈現結果。

## 規則約束
- 輸出結果必須包含雙向連結格式。
- 必須顯示簡單的內容片段以供判讀。
