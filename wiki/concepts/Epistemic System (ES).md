# Epistemic System (認識系統)

## 定義
**Epistemic System (ES)** 是由 [[Carl Winsløw]] (2009) 描述的一個整合性結構，它包含了人類知識產出與習得的所有必要組成部分。它是 **[[GOA-model]]** 所欲描述的客觀對象。

## 組成部分
一個認識系統包含三個互相關聯的子系統：
1.  **具備認知的人類 (G - Humans)**：涉及動機、認知能力、身分認同。
2.  **被認知的對象組織 (O - Organizations)**：涉及數學公理、定義、算法、教學目標。
3.  **認識的中介手段 (A - Artefacts)**：涉及符號系統、科技工具、語言中介。

## 在轉銜研究中的應用
轉銜的「危機」被視為整個 Epistemic System 的不穩定或崩潰。大學要求的 **ES** 與學生中學時期形成的 **ES** 存在根本性的邏輯斷裂。
