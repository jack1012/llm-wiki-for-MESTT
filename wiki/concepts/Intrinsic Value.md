# Intrinsic Value (內在價值)

## 定義
在 **[[Expectancy-Value Theory]]** (EVT) 中，**內在價值** 指的是學生在執行數學任務過程中所獲得的內在樂趣或興趣。

## 在轉銜中的表現 (Geisler, 2023)
- **初始狀態**：新生通常帶著高內在價值進入大學。
- **衰減模式**：遇到挫折（如 **[[Criterial Feedback]]**）後，內在價值會迅速下滑。
- **後果**：當內在價值低於某一閾值時，學生會開始考慮 **[[Early Dropout]]**。
