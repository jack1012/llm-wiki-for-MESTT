# STT 研究卡片：主題地圖與因果鏈

> 本頁面整理了知識庫中 146 張研究卡片的核心想法，分為兩個維度：
> 1. **主題聚類地圖**：按主題群分類，展示每群的核心論點
> 2. **跨主題因果鏈**：從「為什麼」和「如何」的角度，串聯不同主題群之間的因果關係

## Part I：主題聚類地圖

---

### A. 認知斷層與概念理解（~13 張）

**一句話總結**：學生在中學形成的「算法式心智」，與大學要求的「形式化心智」之間存在認識論的根本斷裂。

**核心論點鏈**：
1. 中學數學停留在「感知-符號」世界，大學要求進入「公理-形式」世界 → [[card-three-worlds-of-transition]]
2. 學生依賴的是「概念意象」而非「概念定義」 → [[card-image-vs-definition]]
3. 即使學會了定義，定義在頭腦中常處於「靜止」而非「可操作」狀態 → [[card-the-operability-gap]]
4. 必須完成從「做運算（動詞）」到「談對象（名詞）」的心智轉變 → [[card-objectification-struggle]]
5. 過去在中學的成功，反而形成固化的「事前經驗」，阻礙新學習 → [[card-met-befores-and-cognitive-conflict]]
6. 低層知識的缺口會導致高層概念的「認知崩塌」 → [[card-hierarchical-collapse]]

**教學出口**：
- 反例教學：迫使學生從「怎麼算」轉向「為什麼成立」 → [[card-counter-examples-bridge]]
- D-C-P 循環：讓學生體驗完整的數學活動（定義→猜想→證明） → [[card-d-c-p-activity-cycle]]
- 把定義從「描述性事實」變成「證明性工具」 → [[card-from-problem-solving-to-mathematizing]]

**相關卡片**：
`card-cognitive-rupture-tall`, `card-dtp-teaching-challenge`, `card-fragmented-vs-cohesive`, `card-finding-out-to-understanding`, `card-diagnosis-through-hierarchy`

### B. 情意危機與身分認同（~16 張）

**一句話總結**：轉銜不只是認知挑戰，更是一場身分認同的危機——「我還是那個數學天才嗎？」

**核心論點鏈**：
1. 中學高成就學生的身分認同高度依附於「考試成功」 → [[card-first-failure-experience]]
2. 大學首次失敗引發身分震盪，自問「我是否屬於這裡？」 → [[card-identity-fragility-stt]]
3. 自我概念的崩塌比成績本身更能預測退學 → [[card-self-concept-shatters-dropout]]
4. 態度隨教育階段遞減，大學轉銜期是最低谷 → [[card-attitude-decline-transition]]
5. 情意下滑的**斜率**（前 6-8 週）是退學最強預測因子 → [[card-affective-forecasting-stt]]

**保護因子**：
- 自我效能感作為「心理護盾」，將壓力轉化為動力 → [[card-self-efficacy-as-transition-shield]]
- 「選擇話語」（我主動選擇挑戰）vs「被動話語」（我被迫面對困難），前者培養韌性 → [[card-choice-discourse-in-stt]]
- 教師的效能感會「級聯」傳遞給學生 → [[card-teacher-influence-on-resilience]]

**修復路徑**：
- 轉銜成功 = 在認知衝擊後完成「身分修復」 → [[card-identity-repair-stt]]
- 從「高中專家」退格為「大學新手」的閾限（liminal）階段是必經之路 → [[card-navigating-liminality-identity]]
- 轉銜本質上是「身分工作」——重新定位自己與數學的關係 → [[card-transition-as-identity-work]]

**相關卡片**：
`card-abc-attitude-model`, `card-affective-trajectories-china`, `card-creative-interventions-resilience`, `card-barriers-disabled-stt`, `card-self-advocacy-transition`

### C. 信念、興趣與動機（~14 張）

**一句話總結**：學生帶入大學的「數學觀」決定了他的學習策略、興趣存續、甚至是否退學——信念是整條因果鏈的源頭。

**核心論點鏈**：
1. 數學觀念是學習行為的「決策根源」——觀念不變，策略難改 → [[card-conceptions-drive-strategy]]
2. 靜態信念（數學是固定的規則）vs 動態信念（數學是創造的過程）是核心分野 → [[card-dynamic-beliefs-success]]
3. 動態信念者在轉銜衝擊下能維持滿意度（緩衝作用） → [[card-dynamic-beliefs-as-buffer]]
4. 信念透過**信念→興趣→努力→留存**的連鎖反應影響結局 → [[card-belief-interest-retention-path]]
5. 中學培養的「解題樂趣」≠ 大學需要的「形式化興趣」 → [[card-which-interest-matters]]
6. 興趣錯配不被主動管理，將導致「數學幻滅」 → [[card-interest-mismatch-stt]]

**動機結構**：
- 努力不是「意志力」的問題，而是即時「動機狀態」的反映 → [[card-motivational-drivers-stt]]
- 基本心理需求（勝任感、自主感、關聯感）的滿足決定內在動機 → [[card-needs-satisfaction-mathematics]]
- 學生的「元敘事」（對數學本質的內在敘法）決定挫折後的解讀方式 → [[card-metanarratives-interference]]

**教學出口**：
- 信念是可以透過顯性干預改變的 → [[card-belief-intervention-stt]]
- 職業任務激發情境興趣，預防動機崩潰 → [[card-professional-tasks-as-motivator]]
- 嚴謹性與實用價值需要刻意平衡 → [[card-utility-value-in-transition]]
- 人-環境適配度決定轉銜痛苦程度 → [[card-fit-theory-in-stt]]

**相關卡片**：
`card-interest-trajectories-stt`

### D. 制度結構與課程組織（~16 張）

**一句話總結**：轉銜問題不只是個人失敗，更是中學與大學兩套「知識組織方式」的體制性碰撞。

**核心論點鏈**：
1. 中學數學聚焦 Task + Technique；大學數學聚焦 Technology + Theory → [[card-praxeological-shift-winslow]]
2. 學生試圖用中學的技術去做大學的任務，整個實踐框架不相容 → [[card-praxeological-clash-stt]]
3. Klein 的雙重斷裂不是個人問題，而是體制性的 → [[card-institutional-klein-gap]]
4. 中學考試的「程序共謀」換取了表面高及格率，犧牲了概念嚴謹度 → [[card-procedural-conspiracy]]
5. 中學教學契約在大學環境下徹底崩塌 → [[card-examination-driven-contract]]

**體制的隱形力量**：
- 大學的「弱框架」賦予自主權，卻加劇認知負擔 → [[card-bernstein-framing-stt]]
- 教師期待「結構性反思」，學生執行「程序性計算」 → [[card-university-task-expectations]]
- 隱性課程的解碼能力決定成敗 → [[card-task-misinterpretation-stt]]
- 權力不對等邊緣化新生的「非正規話語」 → [[card-power-dynamics-in-stt]]
- 學生不直接拋棄中學話語，而是讓兩套話語平行並存 → [[card-parallel-discourses-stt]]
- 學生對講師訊息進行「重新語境化」，而非被動接受 → [[card-recontextualization-of-messages]]

**改革的阻力**：
- 體制的自我保護機制抵抗教學創新 → [[card-institutional-inertia-in-reform]]
- 有效教學法在大學大講堂環境下可能「滅絕」 → [[card-atd-ecology-transition]]
- 微積分充當制度性的「守門人」，篩選而非培養 → [[card-calculus-as-gatekeeper]]

**改革的槓桿**：
- 體制必須撥動特定的結構性槓桿（TLC3 框架） → [[card-institutional-levers-transition]]

### E. 教學介入與支持策略（~15 張）

**一句話總結**：有效的轉銜介入不是「補救弱學生」，而是系統性地同時觸及認知支架與情意緩衝。

**四種干預導向**（[[card-stt-intervention-taxonomy]]）：
| 導向 | 核心邏輯 | 代表性介入 |
|------|---------|----------|
| **跳躍** | 幫學生克服認知斷層 | 反例教學, 非常規任務 |
| **涵化** | 引導學生融入大學數學文化 | SRP, 主動學習文化 |
| **多樣性** | 針對不同群體差異化支持 | 診斷測試, MSC |
| **跨社群** | 中學與大學共擔責任 | HCK 培訓, 課程共設計 |

**具體策略**：

*認知層*：
- 非常規任務破除模仿推理 → [[card-non-routine-tasks-catalysts]]
- SRP 將碎片化知識連結為探究式學習 → [[card-srp-as-bridge]], [[card-srp-as-transition-bridge]]
- DDR 作為精準路徑建模的「工程化療法」 → [[card-ddr-as-transition-therapy]]

*情意層*：
- 銜接課程的最大價值是「情意緩衝」而非補教知識 → [[card-bridging-as-affective-buffer]], [[card-bridging-impact-naidoo]]
- 創意情境（戲劇、錄影）降低防禦性，培養韌性 → [[card-creative-interventions-resilience]]
- 敘事寫作在危機前識別高風險學生 → [[card-narrative-diagnostic-tool]]

*制度層*：
- 全方位路徑：入學前 + 中 + 後 → [[card-mcmaster-support-tactics]]
- 低成本每週干預比一次性營隊更有可持續性 → [[card-weekly-intervention-model]]
- MSC 直接關聯留存率 → [[card-msc-impact-retention]]
- 提供支援不夠，還要解決「心理障礙」阻止求助 → [[card-barriers-to-support]]

*文化層*：
- 課堂文化從「排練」到「主動學習」的典範轉移 → [[card-rehearsal-to-active-learning]]
- 生態系統微調 > 單一維度介入 → [[card-transition-as-ecosystem-adjustment]]

### F. 預測因子與預警系統（~15 張）

**一句話總結**：中學成績能預測「考試分數」，但無法預測「是否留下」——後者由情意變量和調適速度驅動。

**預測力的層次**：

*弱預測因子（入學前靜態數據）*：
- 中學 GPA 的盲點：只反映程序熟練度，忽略概念深度 → [[card-conceptual-knowledge-success]]
- 中學積累的學術資產在大一中段後稀釋殆盡 → [[card-marginal-school-influence]]
- 中學成績不是唯一保證，真正的預測因子是「形式演繹準備度」 → [[card-prerequisites-beyond-grades]]

*強預測因子（入學後動態數據）*：
- **調適速度 > 入學高標** → [[card-adaptation-vs-prior-attainment]]
- **期中情意狀態**是最佳預測點，優於入學測驗 → [[card-mid-term-prediction-gold-standard]]
- **自我概念的變化軌跡**比單純成績更能預測退學 → [[card-self-concept-shatters-dropout]]
- **SRL 能力**與終端學業表現強正相關 → [[card-srl-predicts-achievement]]
- **獨立學習能力**是跨越斷層的最強預測因子 → [[card-independent-learning-criticality]]

*退學的直接觸發器*：
- 大一前兩學期的績效要求是最強催化劑 → [[card-performance-as-dropout-trigger]]
- 幻滅學生（高期待→低現實）是退學的「無聲殺手」 → [[card-identifying-disillusioned-students]]

*雙軌評估框架*：
- 成功 = 學業成就 + 心理滿意度，兩者由不同因素驅動 → [[card-dual-track-of-success]], [[card-formal-logic-readiness]]
- 非數學專業存在「高分低能」的數學缺陷 → [[card-math-deficit-bioscience]]

**相關卡片**：
`card-metacognitive-monitoring-stt`, `card-knowledge-facets-stt`, `card-diagnosis-of-underpreparedness`, `card-cognitive-vs-affective-predictors`, `card-affective-forecasting-stt`

### G. 教師視角與師培（~10 張）

**一句話總結**：大學教師不是中立的知識傳遞者——他們的認識論立場、缺失敘事、專業導向，深刻塑造了轉銜的面貌。

**教師的認知地圖**：
1. 教師的底層導向決定其教學決策 → [[card-umt-decision-making]]
2. 多數教師持「缺失敘事」：怪中學、怪學生 → [[card-the-deficit-narrative-among-umts]]
3. 教師的改革思維分為四種取向（跳躍/涵化/多樣性/跨社群） → [[card-four-approaches-to-stt-improvement]]
4. 教師的意識形態影響其採取的干預策略 → [[card-umt-perspectives-on-transition]]

**師培的雙重斷裂**：
- 職前教師在大學經歷第一次斷裂，回到中學經歷第二次斷裂 → [[card-double-discontinuity-pst]]
- 轉銜失敗的教師無法在高等與中學數學之間建立有機連結 → [[card-the-double-discontinuity-paradox]]
- HCK（橫向視野知識）是破解斷裂的最佳路徑 → [[card-hck-as-bridge]]
- 修復關鍵：顯性化高等數學在初等數學中的應用價值 → [[card-connecting-advanced-to-school-math]]

**教師作為改革力量**：
- 優質課程資源能提升教師主體性 → [[card-resources-as-agency-boosters]]
- 教師自我效能感會級聯傳遞給學生韌性 → [[card-teacher-influence-on-resilience]]

### H. 社會文化與脈絡（~9 張）

**一句話總結**：轉銜不只是認知或制度問題——它發生在特定的社會文化脈絡中，受到權力關係、話語規範、文化期待的深刻影響。

**核心論點**：
1. 轉銜本質上是從一種社會實踐切換到另一種 → [[card-math-as-social-practice]]
2. 成功的標誌是「參與」數學社群的專業對話，而非僅僅考試及格 → [[card-participation-in-transition]]
3. 轉銜可被視為一次人類學意義上的「成長儀式」 → [[card-transition-rite-passage]]

**東亞的特殊性**：
- 中學「高效解題型規範」vs 大學「邏輯探究型規範」的社會學碰撞 → [[card-sociomathematical-clash-east-asia]]
- 成功跨越中國轉銜的學生必須從「算法消費者」變為「嚴謹論證者」 → [[card-algorithm-to-rigor-transition]]
- 高度競爭文化使數學身分極度脆弱 → [[card-identity-fragility-stt]]

**權力與話語**：
- 大學課堂的權力不對等邊緣化新生話語 → [[card-power-dynamics-in-stt]]
- 學生用中學話語作為非正規腳手架，逐漸進入大學話語 → [[card-parallel-discourses-stt]]

### I. 理論框架與研究方法（~12 張）

**一句話總結**：STT 研究的理論工具不斷擴充，從單一理論走向「理論聯網」——承認沒有一個框架能獨自解釋轉銜的複雜性。

**理論工具箱**：
- Tall 的三個世界模型 → [[card-three-worlds-of-transition]]
- ATD 的 4T 分析框架 → [[card-praxeological-shift-winslow]], [[card-atd-framework-in-transition]]
- TDS 的微觀教學情境分析 → [[card-tds-atd-lenses]]
- 三維度障礙分類（認識論-教學法-個體發生） → [[card-the-three-dimensions-of-obstacles]]
- 四平面模型（個人-微觀社會-機構-文化歷史） → [[card-four-planes-of-transition]]

**研究轉向**：
- 從認知缺陷 → 整全體驗的範式轉換 → [[card-from-cognitive-to-holistic]]
- 2008-2021 年：轉向情境關注與學生多樣性 → [[card-stt-research-trends-post-2008]]
- 2016 年：從「學生認知」轉向「體制與社會文化」 → [[card-tertiary-math-trends-2016]]
- ICME-13 確認了理論碎片化的現實 → [[card-icme13-transition-map]]

**方法論整合**：
- 理論聯網建立「跨邊界對話」，不追求統一 → [[card-praxis-of-networking]]
- 單一維度介入通常無效，需要生態系統調適 → [[card-transition-as-ecosystem-adjustment]]

### J. 退學與留存（~7 張）

**一句話總結**：退學不是個人意志力的失敗，而是「課程難度 × 動機錯位 × 制度篩選」交織的結構性危機。

**退學的結構**：
- 課程難度 + 動機錯位 = 結構性危機 → [[card-macro-causes-of-math-dropout]]
- 微積分作為制度性守門人，篩選而非培養 → [[card-calculus-as-gatekeeper]]
- 績效要求是直接催化劑，背後是認知轉移失敗 → [[card-performance-as-dropout-trigger]]
- 幻滅學生（高期待→低現實）是無聲殺手 → [[card-identifying-disillusioned-students]]
- 知識僵化導致跨領域遷移失敗 → [[card-transfer-difficulty-stt]]

### 因果鏈 1：「一個好學生如何走向退學」

這是知識庫中證據最豐富的一條因果鏈，串聯了 A → D → B → J 四個主題群：

```
中學的「程序共謀」             大學的「認知斷裂」
(card-procedural-conspiracy)  →  (card-cognitive-rupture-tall)
         ↓                              ↓
形成不適合大學的概念意象         DTP 教學模式形成「邏輯之牆」
(card-image-vs-definition)      (card-dtp-teaching-challenge)
                                         ↓
                                首次考試失敗
                               (card-first-failure-experience)
                                         ↓
                               自我概念崩塌
                              (card-self-concept-shatters-dropout)
                                         ↓
                               情意軌跡急速下滑
                              (card-affective-forecasting-stt)
                                         ↓
                               退學決定
                              (card-performance-as-dropout-trigger)
```

**延伸支線**：
- 中學興趣 ≠ 大學成功 → 興趣錯配 → 幻滅 → 退學
  `(card-which-interest-matters) → (card-interest-mismatch-stt) → (card-identifying-disillusioned-students)`

### 因果鏈 2：「信念如何驅動留存或退學」

串聯 C → B → J：

```
中學培養的數學觀念
(card-conceptions-drive-strategy)
         ↓
    ┌────┴────┐
  靜態信念     動態信念
(數學=固定規則)  (數學=創造過程)
    ↓              ↓
面對大學衝擊     面對大學衝擊
```

## 📌 整理的核心發現

### 1. 卡片之間不是獨立的——它們構成因果網絡

146 張卡片的想法並非零散的研究發現。它們之間存在清晰的因果邏輯：
- **上游因素**（中學的程序共謀、靜態信念、制度設計）決定了**下游結果**（退學、留存、滿意度）
- **保護因子**（動態信念、自我效能、SRL）可以中介和緩衝上游因素的負面影響
- **改革的阻力**（教師缺失敘事、體制慣性）解釋了為什麼問題持續循環

### 2. 六條因果鏈的交匯點

所有因果鏈最終匯聚在兩個關鍵節點：
1. **身分修復**（[[card-identity-repair-stt]]）：學生能否在認知衝擊後重建「我是數學人」的認同
2. **制度槓桿**（[[card-institutional-levers-transition]]）：大學是否願意從結構層面改變，而非只要求學生適應

### 3. 介入的「黃金窗口」

從因果鏈的時間維度看：
- **最早的介入點**：信念（可以在入學前或銜接課程中干預）
- **最關鍵的預警窗口**：大一第 3-8 週
- **最深層的改變**：教師師培中打破雙重斷裂
